# INDICADOR-INTELIGENTE
"Indicador de trading inteligente con IA para Forex, basado en datos de mercado real (OANDA). Diseñado para escanear múltiples activos, identificar señales proactivas de alta probabilidad y determinar el vencimiento óptimo (entre 2 y 5 minutos) utilizando un análisis avanzado de fuerza y volumen."
